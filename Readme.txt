Scuba Adventure is a underwater puzzle game with original game physics. 
Drag and drop bubble makers to push your character and collect the treasure! 
There are 3 levels and a create your own level feature.

Demo Video: https://youtu.be/nWxMoCGJULs

How to Run:
Make sure pygame is installed
Make sure all image files are in the same folder as the game 
Run Scuba Adventure.py
